# 2015 Air Mint XFCE Tweaks

## Project Overview
Running Linux Mint XFCE on a 2015 Macbook Air 1.6ghz with 8gb RAM and 256gb SSD.

## Tweaks
1) System wakes from sleep on its own after a few seconds when the lid is open. Mitigated by disabling lid as a wake up trigger via disable-lid-wake.service

2) BattleTech inspired conky readout via conky.conf
